title: Tablas - Usuarios
description: Tablas del módulo de usuarios
---

## usuarios

| Campo | Tipo | Descripción |
|-------|------|-------------|
| id | int | Identificador único |
| nombre | varchar(100) | Nombre completo del usuario |
| email | varchar(150) | Correo electrónico |
| password | varchar(255) | Contraseña hasheada |
| rol_id | int | ID del rol asignado |
| created_at | timestamp | Fecha de creación |
| updated_at | timestamp | Fecha de última modificación |
| active | boolean | Estado del usuario |

## roles

| Campo | Tipo | Descripción |
|-------|------|-------------|
| id | int | Identificador único |
| nombre | varchar(50) | Nombre del rol |
| descripcion | text | Descripción del rol |
| permisos | json | Permisos asignados al rol |
| created_at | timestamp | Fecha de creación |
| active | boolean | Estado del rol |

## sesiones

| Campo | Tipo | Descripción |
|-------|------|-------------|
| id | varchar(255) | ID de la sesión |
| usuario_id | int | ID del usuario |
| ip_address | varchar(45) | Dirección IP |
| user_agent | text | Información del navegador |
| created_at | timestamp | Fecha de inicio |
| expires_at | timestamp | Fecha de expiración |
| active | boolean | Estado de la sesión |
