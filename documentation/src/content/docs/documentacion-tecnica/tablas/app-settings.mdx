# App Settings

## configurations

| Campo | Tipo | Descripción |
|-------|------|-------------|
| id | UUID | Identificador único |
| key | VARCHAR | Clave de configuración única |
| value | TEXT | Valor de configuración |
| type | ENUM | Tipo de valor |
| is_encrypted | BOOLEAN | Valor encriptado |
| description | TEXT | Descripción de la configuración |
| created_at | TIMESTAMP | Fecha de creación |
| updated_at | TIMESTAMP | Fecha de modificación |
