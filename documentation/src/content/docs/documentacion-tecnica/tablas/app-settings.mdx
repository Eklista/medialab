# App Settings

## configurations

| Campo | Tipo | Descripción |
|-------|------|-------------|
| id | UUID | Identificador único |
| key | VARCHAR | Clave de configuración única |
| value | TEXT | Valor de configuración |
| type | ENUM | Tipo de valor |
| is_encrypted | BOOLEAN | Valor encriptado |
| description | TEXT | Descripción de la configuración |
| created_at | TIMESTAMP | Fecha de creación |
| updated_at | TIMESTAMP | Fecha de modificación |

## configuration_history

| Campo | Tipo | Descripción |
|-------|------|-------------|
| id | UUID | Identificador único |
| configuration_id | UUID | ID de la configuración |
| changed_by_user_id | UUID | ID del usuario que hizo el cambio |
| old_value | TEXT | Valor anterior |
| new_value | TEXT | Valor nuevo |
| change_reason | TEXT | Razón del cambio |
| change_type | ENUM | Tipo de cambio |
| requires_restart | BOOLEAN | Requiere reinicio del sistema |
| approved_by_user_id | UUID | ID del usuario que aprobó |
| approved_at | TIMESTAMP | Fecha de aprobación |
| rollback_available | BOOLEAN | Se puede hacer rollback |
| created_at | TIMESTAMP | Fecha del cambio |

### Relaciones
- **configuration_id** → configurations.id
- **changed_by_user_id** → users.id
- **approved_by_user_id** → users.id

### Tipos de cambio:
- **UPDATE** → Actualización de valor
- **DELETE** → Eliminación de configuración
- **CREATE** → Creación de nueva configuración
- **ROLLBACK** → Rollback a versión anterior

## project_templates

| Campo | Tipo | Descripción |
|-------|------|-------------|
| id | UUID | Identificador único |
| name | VARCHAR | Nombre del template |
| description | TEXT | Descripción del template |
| created_by_user_id | UUID | ID del usuario creador |
| service_type_id | UUID | ID del tipo de servicio |
| default_config | JSONB | Configuración por defecto |
| variable_fields | JSONB | Campos que el usuario debe llenar |
| assigned_users | JSONB | Usuarios que pueden usar este template |
| is_active | BOOLEAN | Template activo |
| created_at | TIMESTAMP | Fecha de creación |
| updated_at | TIMESTAMP | Fecha de modificación |

### Relaciones
- **created_by_user_id** → users.id
- **service_type_id** → service_types.id

### Ejemplo de default_config:
```json
{
  "duration_hours": 2,
  "location": "Auditorio Principal",
  "equipment_needed": ["CAMARA_1", "CAMARA_2"],
  "default_assignees": {
    "cameraman": "juan_garcia_id",
    "tech_support": "ana_lopez_id"
  }
}
```

### Ejemplo de variable_fields:
```json
{
  "required_fields": [
    {"field": "event_date", "type": "date", "label": "Fecha del evento"},
    {"field": "main_topic", "type": "text", "label": "Tema principal"}
  ],
  "optional_fields": [
    {"field": "vip_guests", "type": "text", "label": "Invitados VIP"}
  ]
}
```
