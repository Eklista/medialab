# Projects (General)

## projects

| Campo | Tipo | Descripción |
|-------|------|-------------|
| id | UUID | Identificador único |
| project_type_id | UUID | ID del tipo de proyecto |
| name | VARCHAR | Nombre del proyecto |
| description | TEXT | Descripción del proyecto |
| client_id | UUID | ID del cliente |
| unit_id | UUID | ID de la unidad |
| status_id | UUID | ID del estado |
| priority_id | UUID | ID de la prioridad |
| request_date | DATE | Fecha de solicitud |
| start_date | DATE | Fecha de inicio |
| end_date | DATE | Fecha de finalización |
| observations | TEXT | Observaciones |
| created_at | TIMESTAMP | Fecha de creación |
| updated_at | TIMESTAMP | Fecha de modificación |

### Relaciones
- **project_type_id** → project_types.id
- **client_id** → users.id
- **unit_id** → units.id
- **status_id** → status_options.id
- **priority_id** → priority_options.id
