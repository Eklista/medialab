# Tasks & Workflow

## tasks

| Campo | Tipo | Descripción |
|-------|------|-------------|
| id | UUID | Identificador único |
| project_id | UUID | ID del proyecto |
| podcast_episode_id | UUID | ID del episodio de podcast |
| class_id | UUID | ID de la clase |
| service_type_id | UUID | ID del tipo de servicio |
| status_id | UUID | ID del estado |
| priority_id | UUID | ID de la prioridad |
| name | VARCHAR | Nombre de la tarea |
| description | TEXT | Descripción de la tarea |
| delivery_date | DATE | Fecha de entrega |
| observations | TEXT | Observaciones |
| created_at | TIMESTAMP | Fecha de creación |
| updated_at | TIMESTAMP | Fecha de modificación |

### Relaciones
- **project_id** → projects.id
- **podcast_episode_id** → podcast_episodes.id
- **class_id** → classes.id
- **service_type_id** → service_types.id
- **status_id** → status_options.id
- **priority_id** → priority_options.id
